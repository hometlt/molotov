Использование

Ставить в секцию <head> страницы, на которую будет установлен плагин следующий код

    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <script type="text/javascript" src="http://molotov.hometlt.ru/sprite.js"></script>
    <link type="text/css" rel="stylesheet" media="all" href="http://molotov.hometlt.ru/sprite.css"/>

в место, где хотите установить изобрадение следующую строку

<div id="sprite1"></div>
<div id="sprite2"></div>

Затем, следующий код 

<script type="text/javascript">
    $("#sprite1").sprite({
        images: "../assets/Photo-{1-23}.jpg",
        responsive: true, //по умолчанию false
        width: 900,//по умолчанию 900
        height: 600
    });
</script>

3 способа перечисления фотографий

  • при помощи массива 
     images: ["../assets/Photo-1.jpg","../assets/Photo-2.jpg"]
  • Использование "{}" для перечисляния группы фотографий с похожими названиями 
     images: "../assets/Photo-{1-23}.jpg",
  • Использование "{}" для перечисляния отждельных фотографий 
     images: "../assets/Photo-{1,4,8}.jpg",


2 вариант

javascript

 
        $(".molotov-sprite").sprite();

html

<div class="molotov-sprite" data-images="../assets/Photo-{2,4,6,8,10}.jpg"  data-height=300 data-width=450></div>
<div class="molotov-sprite" data-images="../assets/Photo-{6-23}.jpg" data-height=600></div>
<div class="molotov-sprite" data-images="{unexisted1.jpg,unexisted2.jpg}" data-height=600></div>
